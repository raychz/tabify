# Tabify

## Getting Started
```js
npm install -g ionic
ionic serve
```

## Running on Platform Emulator or Connected Device
```js
ionic cordova run [<platform>] [options]
// Example:
ionic cordova run ios -lc --target="iPhone-6s"
```
